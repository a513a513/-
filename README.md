# 🎮 숫자 맞추기 게임 (Up & Down Game)

Java Swing을 이용하여 개발한 GUI 기반의 **숫자 맞추기 게임**입니다.
객체지향 설계를 준수하여 기능과 화면을 분리하였으며, 다양한 예외 처리를 통해 완성도를 높였습니다.

## 📋 프로젝트 개요
* **프로젝트명:** Up & Down 숫자 맞추기 게임
* **개발 환경:** Java JDK, IntelliJ IDEA
* **개발자:** [박준홍/2021963034]

## ✨ 주요 기능
1.  **게임 로직**
    * 1~100 사이의 랜덤 숫자가 정답으로 생성됩니다.
    * 사용자의 입력값에 따라 **Up / Down / 정답** 결과를 실시간 로그로 보여줍니다.
    * 시도 횟수(Count)가 기록됩니다.

2.  **예외 처리**
    * **숫자 외 입력 방지:** 문자를 입력하면 경고창(`JOptionPane`)을 띄우고 입력창을 초기화합니다.
    * **범위 제한:** 1 미만, 100 초과 숫자 입력 시 경고창을 띄워 게임의 오류를 방지합니다.
    * **입력 편의성:** 오류 발생 후 자동으로 입력창을 비우고 포커스를 이동시켜 바로 다시 입력할 수 있습니다.

3.  **게임 재시작**
    * 정답을 맞히면 '확인' 버튼이 비활성화되고, '재시작' 버튼이 활성화되어 즉시 새 게임을 시작할 수 있습니다.

## 🛠 객체지향 설계
클래스를 역할별로 분리하였습니다.

### 1. Main.java
* **역할:** 프로그램의 진입점(Entry Point).
* **특징:** `SwingUtilities.invokeLater`를 사용하여 스윙 스레드의 안전성을 확보하며 GUI를 실행합니다.

### 2. GameFrame.java
* **역할:** 화면(View) 구성 및 게임 로직(Controller) 담당.
* **구성 요소:**
    * `initGame()`: 난수 생성 및 UI 초기화 메소드.
    * `checkNumber()`: 사용자의 입력값을 검증하고 정답과 비교하는 핵심 로직.
    * **GUI 컴포넌트:** `JFrame`, `JPanel`, `JButton`, `JTextField`, `JTextArea`, `JScrollPane` 사용.
